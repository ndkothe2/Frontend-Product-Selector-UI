Interactive E-commerce Product Selector (Tiered Discount / BOGO)

This document provides an overview of an interactive product selector component built to showcase tiered e-commerce purchasing options. This single-page frontend component allows customers to choose between different unit quantities or bundles, displaying associated discounts and dynamic customization options.

Key Features:

Intuitive Interactive Selection: Users can effortlessly select their preferred package (1 Unit, 2 Unit, or 3 Unit) with a simple click anywhere on the product box, ensuring a smooth user experience.
Immediate Visual Feedback: The component utilizes Visual State Management where the selected bundle is immediately highlighted with a bold red border and a soft pink background, providing instant and clear feedback to the customer.

Seamless Dynamic Customization: Upon selection, a detailed configuration panel smoothly expands, allowing customers to specify Size and Colour for every individual unit within the chosen package.
Transparent Pricing Display: Each option clearly presents the discounted current price alongside the original price, utilizing a striking strikethrough effect for maximum transparency.
Prominent Discount Highlighting: Attractive Percentage Discount Tags (10% OFF, 20% OFF, 30% OFF) are prominently featured, clearly communicating the savings to the user.

Checkout Optimization: A "Most Popular" ribbon tag is used strategically to highlight the recommended bundle (2 Unit), guiding customer decision-making.
Real-Time Total Summary: The final checkout summary updates its total price instantly and dynamically, ensuring the customer always sees the correct cost for the selected deal.

Contextual Call-to-Action: The "Add to Cart" Button is dynamic, updating its label to reflect the selected package (e.g., "Add 2 Unit deal to cart"), enhancing clarity and conversion potential.
Modern Aesthetic Design: The entire component boasts a clean layout, professional typography, and a cohesive pink/red color palette for a high-quality visual presentation.
